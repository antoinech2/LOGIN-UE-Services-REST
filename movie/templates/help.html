Help page

There is the list of available endpoints :

  • GET / - Homescreen
  • GET /template - HTML template example
  • GET /json - Raw JSON Database of movies
  • GET /movies/<id> - Retrieve a specific movie by ID
  • GET /movies/<id>/rating - Retrieve a specific movie rating by ID
  • GET /movies/<id>/title - Retrieve a specific movie title by ID
  • GET /movies/<id>/director - Retrieve a specific movie director by ID
  • GET /moviesbytitle?title=<title> - Retrieve a specific movie by its title
  • POST /addmovie/<id> - Add a new movie in database. See documentation.
  • PUT /movies/<id>/<rate> - Update a movie rating by ID
  • DELETE /movies/<id> - Delete a movie by ID
  • GET /help - Show this page
